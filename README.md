# Geek Salles System (GSS)

O Geek Salles System (GSS) é um protótipo de sistema desenvolvido com o objetivo de criar uma solução voltada para a gestão da GEEK’s Store, uma loja fictícia especializada em jogos, acessórios e produtos do universo geek.

## **Objetivo**

Desenvolver um sistema de controle de vendas e estoque capaz de atender às necessidades reais de um comércio geek, otimizando processos administrativos e melhorando o controle operacional.

- Documentação - Requisitos, casos de uso, diagrama de atividades e modelo de entidade e relacionamento
- Protótipos  - Telas da aplicação
