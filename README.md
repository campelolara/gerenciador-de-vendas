# Geek Salles System (GSS)

O Geek Salles System (GSS) é um protótipo de sistema desenvolvido com o objetivo de criar uma solução voltada para a gestão da GEEK’s Store, uma loja fictícia especializada em jogos, acessórios e produtos do universo geek.

## **Funcionalidades**

- Gerenciamento de vendas
- Cadastro de produtos e equipamentos
- Registro de clientes e funcionários
- Controle de estoque
